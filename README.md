# Anyactor PRoot System 🚀

A fast, highly optimized, lightweight, and modern script to deploy an isolated **Ubuntu 22.04 LTS** virtual environment inside non-root environments (like Termux, specialized hosting panels, or minimal Linux installations) using PRoot.

---

## ✨ Features

* **Zero Root Requirements:** Runs completely in user space without requiring `sudo` or root permissions.
* **Architecture Detection:** Automatically identifies and configures environments for both `x86_64` (amd64) and `aarch64` (arm64) architectures.
* **Security Guard:** Features a safety interlocking prompt that halts unauthorized execution until explicitly authorized.
* **Immersive Visuals:** Built-in terminal matrix sequences provide immediate visual feedback during compilation and payload injection.
* **Automated Guest Setup:** Comes pre-packaged with an internal system updater that automatically provisions core utilities (`python3`, `pip`, `git`, `curl`, `tmux`, `ssh`) on its initial run.
